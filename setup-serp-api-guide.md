# Bright Data SERP API のセットアップ

### Step 1: ダッシュボードにアクセスする
- Bright Data アカウントに[ログイン](https://brightdata.jp/cp/start)します
- 左側メニューで「Proxies & Scraping」に移動します
- 「SERP API」を選択し、「Get Started」をクリックします

<img width="800" alt="bright-data-serp-api-proxies-scraping-infrastructure" src="https://github.com/user-attachments/assets/5cfb7f6c-6043-46c2-acd2-60f352907f8f" />

### Step 2: Zone を作成する
- 分かりやすい Zone 名を入力します（作成後は変更できません）
- 「Add」をクリックします
- ポップアップダイアログで Zone の作成を確認します

<img width="800" alt="bright-data-add-new-serp-api-zone" src="https://github.com/user-attachments/assets/e72f67ea-9b82-413d-82cc-fc1027e0a623" />

### Step 3: 認証情報にアクセスする
- 作成が完了すると、Overview ページに移動します
- 2つのタブで認証情報を確認します:
    - Direct API Access
    - Native Proxy-Based Access
